# 🎬 Douban Lite (豆瓣精简版)

> 一个基于原生 HTML/CSS/JS 构建的纯净版豆瓣前端演示项目。去除了冗余的广告与社交功能，回归“记录生活、分享热爱”的评分初心。

## 📖 项目介绍

**Douban Lite** 是一个多页面的静态 Web 应用（MPA）。它完全不依赖任何现代前端框架（如 Vue/React）或打包工具，仅使用原生技术栈实现。

尽管没有后端数据库，但项目通过精心设计的 **JavaScript 逻辑** 和 **JSON 数据模拟**，实现了高度动态的交互体验，包括全站搜索、详情页路由、评论交互等功能。

## ✨ 核心功能

1.  **智能数据加载 (Smart Fetching)**
    *   优先尝试通过 `fetch API` 读取 `db.json` 文件。
    *   **容错机制**：如果直接通过文件系统打开（`file://` 协议导致跨域）或网络请求失败，会自动降级使用脚本内置的 `FALLBACK_DATA`，确保页面永远不会空白。
2.  **动态路由与渲染**
    *   单脚本驱动：所有页面共用 `script.js`，脚本根据 DOM 钩子自动判断当前页面并执行相应逻辑。
    *   详情页渲染：通过解析 URL 参数（如 `detail.html?type=movie&id=1`）动态加载不同的电影或图书详情。
3.  **搜索与过滤系统**
    *   **全站搜索**：首页支持实时检索电影标题、导演、图书作者等。
    *   **局部筛选**：频道页支持通过标签（Tag）点击过滤内容，以及局部关键字搜索。
4.  **交互演示**
    *   **评论功能**：详情页支持添加评论，并带有 DOM 插入动画。
    *   **模拟登录**：包含一个简单的登录表单验证逻辑。
5.  **现代化 UI 设计**
    *   使用 CSS Grid 和 Flexbox 布局。
    *   CSS 变量（Variables）管理主题色。
    *   固定头部导航（Fixed Header）与防遮挡处理。

## 🛠 技术栈

*   **HTML5**: 语义化标签结构。
*   **CSS3**: 
    *   Grid 网格布局 (首页、列表页)
    *   Flex 弹性布局 (导航、卡片详情)
    *   CSS Animation (评论淡入效果)
*   **JavaScript (ES6+)**:
    *   `async/await` 异步数据处理
    *   `URLSearchParams` 路由参数解析
    *   DOM 操作与事件委托

## 🚀 如何运行

本项目是纯静态的，你可以通过以下两种方式运行：

### 方式 A：直接打开 (最简单)
双击文件夹中的 `index.html`。
*   *注意：此时控制台可能会报 CORS 错误，但得益于项目的 **Fallback 机制**，页面依然可以完美运行。*

### 方式 B：使用 Live Server (推荐)
为了体验完整的 `fetch` 请求流程，建议使用本地服务器环境。
1.  使用 **VS Code** 打开项目文件夹。
2.  安装 **Live Server** 插件。
3.  在 `index.html` 上右键点击 "Open with Live Server"。

## 📝 学习重点

如果你是前端初学者，这个项目值得关注的代码片段：

1.  **`script.js` 中的 `loadData()` 函数**: 学习如何优雅地处理异步请求失败。
2.  **`detail.html` 的渲染逻辑**: 学习如何不使用框架，仅靠 URL 参数实现“一个 HTML 文件展示无数个商品”。
3.  **`style.css` 中的 `.search-box`**: 学习如何使用 `transform: translate(-50%, -50%)` 实现绝对居中。

# 🎥 Douban Lite - 泛文化兴趣社区

> \*\*Web 应用开发课程设计项目\*\*  
> 一个基于原生 HTML/CSS/JavaScript 构建的轻量级仿豆瓣网应用。

## 📖 项目简介

**Douban Lite** 是一个集电影、图书、小组话题于一体的静态 Web 应用。本项目旨在通过原生前端技术还原豆瓣网的核心浏览体验。

项目采用 **JSON 数据驱动** 的开发模式，模拟了前后端分离的数据交互流程。用户可以浏览热门内容、通过标签筛选兴趣条目、进行全站搜索、查看详细信息并发表评论。

## 🛠️ 技术栈

* **HTML5**: 语义化标签构建页面结构。
* **CSS3**: Flexbox \& Grid 布局，实现响应式网格与美观的 UI。
* **JavaScript (ES6+)**: 处理 DOM 操作、事件监听、URL 参数解析。
* **Fetch API**: 异步读取 `db.json` 本地数据，模拟 API 请求。
* **JSON**: 存储电影、图书、小组等结构化数据。

## ✨ 核心功能

1. **多页面架构**：包含聚合首页、电影频道、图书频道、小组列表、内容详情页、登录页。
2. **动态数据渲染**：所有内容卡片均通过 JS 读取 `db.json` 动态生成，非硬编码。
3. **全站与频道搜索**：

   * 首页支持同时搜索电影和图书。
   * 频道页支持当前类目的精准搜索。

4. **标签筛选系统**：点击侧边栏标签（如“科幻”、“文学”），实时过滤展示内容。
5. **详情页路由**：通过 URL 参数（如 `?type=movie\&id=1`）复用同一个 HTML 模板展示不同内容。
6. **模拟交互**：

   * **评论功能**：详情页支持发表短评（DOM 节点动态插入）。
   * **登录验证**：实现了基础的账号密码验证逻辑。

## 🚀 如何运行 (非常重要)

由于项目使用了 `Fetch API` 读取本地 JSON 文件，受浏览器 **CORS (跨域资源共享)** 策略限制，**不能直接双击 HTML 文件打开**。

### 方法一：使用 VS Code Live Server (推荐)

1. 在 VS Code 中安装插件 **"Live Server"**。
2. 在编辑器中右键点击 `index.html`。
3. 选择 **"Open with Live Server"**。
4. 浏览器将自动打开 `http://127.0.0.1:5500/index.html`，项目即可正常运行。

### 方法二：部署到静态托管服务

将项目文件夹上传至 **GitHub Pages**、**Gitee Pages** 或 **Netlify**，即可直接访问。
